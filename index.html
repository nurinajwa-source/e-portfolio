Career Exploration

Chosen career role: Software Developer with security awareness.

Why I Choose Software Developer

I choose Software Developer because I enjoy building systems that can solve real problems. I am interested in creating websites, mobile applications, and user-friendly systems. Since software is used in many industries, this career gives me the chance to keep learning and improve my technical skills.

Job Responsibilities

  • Design and develop web or mobile applications.
  • Write clean and functional coding.
  • Connect systems with databases such as MySQL and Firebase.
  • Test and fix errors in the system.
  • Improve system security, user login, and data protection.

Required Skills

  • Programming knowledge such as PHP, HTML, CSS, JavaScript, and Dart.
  • Database management using MySQL, Firebase, or Firestore.
  • Basic networking and cybersecurity knowledge.
  • Problem-solving and logical thinking.
  • Teamwork and communication skills.

Relevance to Cybersecurity

Software developers must understand cybersecurity because applications handle user data, login information, and private records. A developer should know how to prevent weak passwords, insecure databases, poor access control, and unsafe coding practices. This is important to protect users and build trustworthy systems.
