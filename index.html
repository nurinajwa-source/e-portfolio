01 / CAREER EXPLORATION

Software Developer

I choose Software Developer as my career role because I enjoy creating systems, solving problems, and building applications that can help users.

Why I Choose This Career

I choose Software Developer because I like to turn ideas into real working systems. This career suits me because it combines logic, creativity, design, and problem-solving. I am also interested in web and mobile application development.

Job Responsibilities

  • Develop web and mobile applications.
  • Write clean and understandable code.
  • Test, debug, and improve system functions.
  • Connect applications with databases.
  • Apply basic security in login and data handling.

Required Skills

  • Programming languages such as PHP, JavaScript, and Dart.
  • Database knowledge such as MySQL and Firebase.
  • Basic cybersecurity awareness.
  • UI/UX design understanding.
  • Problem-solving and teamwork skills.

Relevance to Cybersecurity

Software developers need cybersecurity knowledge because applications usually handle user data, passwords, and private information. Secure coding helps prevent weak login systems, data leakage, and unauthorized access.
