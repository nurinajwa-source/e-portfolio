01. CAREER EXPLORATION

Chosen Career Role

The career role that I choose is Software Developer with security awareness.

// Why Software Developer?

I choose Software Developer because I enjoy creating systems and applications that can solve real problems. I am interested in building websites, mobile apps, and database systems. This career is suitable for me because it allows me to use creativity, logic, and technical skills together.

// Job Responsibilities

  • Design and develop web or mobile applications.
  • Write, test, and debug coding.
  • Connect applications with databases.
  • Improve system performance and user experience.
  • Apply secure coding practices to protect user data.

// Required Skills

  • Programming knowledge such as HTML, CSS, JavaScript, PHP, and Dart.
  • Database skills using MySQL, Firebase, or Firestore.
  • Understanding of system design and user interface.
  • Basic networking and cybersecurity knowledge.
  • Problem-solving, teamwork, and communication skills.

// Relevance to Cybersecurity

Software development is related to cybersecurity because applications store user information, login details, and private data. A developer must understand secure login, access control, password protection, and safe database handling. This helps prevent data leakage and protects users from cyber threats.
