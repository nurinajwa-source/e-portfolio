01 / CAREER EXPLORATION

Software Developer

I choose Software Developer because I enjoy creating systems, solving problems, and turning ideas into real applications that users can use.

Why I Choose This Career

This career matches my interest because it combines logic, creativity, design, and problem-solving. I like building websites, mobile apps, and systems that can make tasks easier for users.

Main Responsibilities

  • Develop web and mobile applications.
  • Write clean and understandable code.
  • Test, debug, and improve system functions.
  • Connect systems with databases and APIs.
  • Apply basic security in login and data handling.

Required Skills

  • Programming and problem-solving skills.
  • Database knowledge such as MySQL and Firebase.
  • Basic cybersecurity awareness.
  • UI/UX design understanding.
  • Teamwork and communication skills.

Cybersecurity Relevance

Software developers need cybersecurity knowledge because most systems handle user data, passwords, and private information. Secure coding helps reduce risks such as weak login, data leakage, and unauthorized access.
